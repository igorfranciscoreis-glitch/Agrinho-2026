# Agrinho-2026 Diário de Campo Agrotech
Diário de Campo Agrotech: uma plataforma web criada com HTML, CSS e JS para gestão agrícola. Permite mapear lotes, culturas, prioridades e rotinas (plantio, colheita, manejo). Possui painel de estatísticas, filtros avançados e usa localStorage para salvar os dados no navegador, eliminando o papel e organizando a lavoura pelo celular.
